# Loja de Peças para PCs Gamers

Bem-vindo à Loja de Peças para PCs Gamers! Este é um site simples e responsivo para venda de componentes para PCs gamers, desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- **Introdução animada**: Tela de boas-vindas com animações de fade-in.
- **Produtos em destaque**: Lista de peças como processadores, placas de vídeo, RAM, SSD e notebooks.
- **Carrinho de compras**: Adicione e remova itens, com cálculo automático do total.
- **Responsivo**: Design adaptável para desktop, tablet e mobile.
- **Contato**: Informações de contato.

## Tecnologias Usadas

- **HTML5**: Estrutura do site.
- **CSS3**: Estilos, animações e responsividade.
- **JavaScript**: Funcionalidades interativas (carrinho, navegação).
- **Jekyll**: Para build e hospedagem no GitHub Pages.

## Como Usar

1. Clone o repositório: `git clone https://github.com/Carlos2007Henrique/primeiro---Site.git`
2. Abra `index.html` no navegador ou use Jekyll para build.
3. Para desenvolvimento local: Instale Jekyll e rode `jekyll serve`.
